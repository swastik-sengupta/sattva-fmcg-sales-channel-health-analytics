# Sattva Foods FMCG Sales & Distribution Analytics

**Video Walkthrough:** [Link to LinkedIn Video will go here]

## The Scenario
Sattva Foods is a natural staples brand targeting Tier-2 and Tier-3 Kirana stores in Eastern India. This project analyzes three years of secondary sales data to identify supply chain bottlenecks, localized product-market fit failures, and sales representative inefficiencies.

## The Tech Stack
* Python (Pandas, Faker) for synthetic data engineering
* Supabase (PostgreSQL) for cloud data warehousing
* DBeaver for SQL scripting and View creation
* Power BI for dashboarding and geographic analysis
